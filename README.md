# 🎓 Student Data Organizer

A simple Python project to manage student records using **List, Tuple, Set, and Dictionary** concepts. This project allows users to add, display, update, and delete student information.

---

## 📌 Features

✅ Add Student Details
✅ Display All Students
✅ Update Student Age
✅ Delete Student Records
✅ Display All Subjects
✅ Menu-Driven Program
✅ Uses Python Collections (List, Tuple, Set, Dictionary)

---

## 🛠️ Concepts Used

* Variables
* Input/Output
* Type Casting
* if-elif-else
* while Loop
* for Loop
* List
* Tuple
* Set
* Dictionary
* String Functions
* del Keyword

---

## 📂 Student Information Stored

Each student record contains:

* Student ID
* Date of Birth
* Name
* Age
* Grade
* Subjects

---

## ▶️ How to Run

1. Install Python.
2. Save the code as `student_data_organizer.py`
3. Open Terminal or Command Prompt.
4. Run the program:

```bash
python student_data_organizer.py
```

---

## 📋 Menu Options

```
1. Add Student
2. Display Students
3. Update Student Age
4. Delete Student
5. Display Subjects
6. Exit
```
